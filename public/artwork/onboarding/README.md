# Onboarding illustrations (Phase 27)

## Files

| File | Scene |
|------|--------|
| `onboarding-welcome.svg` | Welcome — fanned cards, soft glow, horizon line |
| `onboarding-add-binder.svg` | Add binder — spine + rings + emerald plus badge |
| `onboarding-scan-card.svg` | Scan — device frame, card, scan line, corner brackets |
| `onboarding-start-trading.svg` | Trading — two tilted cards + swap arrow |

All use **`viewBox="0 0 520 280"`** (wide scene card) so they scale from narrow mobile steps to ~1024px layouts.

## PNG export

1. Open any `.svg` in Figma, Illustrator, or Inkscape.
2. Export at **1040×560** (@2x) or **520×280** (@1x) for raster fallbacks.
3. Prefer **SVG** in the app for crisp scaling.

## Style

Aligned with `ARTWORK_STYLE_GUIDE.md` and Phase 25 marketing language: zinc surfaces, emerald accents, amber highlights, soft rounded geometry, no photographic textures, no copy inside the SVG (headlines live in React).
