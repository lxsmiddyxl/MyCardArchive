# Roadmap (rolling)

Priorities shift with user feedback; this is a working snapshot for engineering alignment.

## Near term

- Deeper **trade** tooling (counter-offers, clearer state transitions).
- **Scan** quality: model configuration, better empty/match-fail copy.
- **Grading** model swap path when a trained endpoint is ready.

## Medium term

- Richer **binder** sharing (read-only views without weakening RLS).
- **Deck** meta: tournament notes, sideboard guides (opt-in).
- **Analytics** surfaces for collection value trends (privacy-preserving).

## Ongoing

- Performance: virtual lists, fewer waterfalls, edge caching where safe.
- Accessibility: focus order, labels, live regions on async flows.
- Operations: rate limits, health endpoints, telemetry SLOs (see `docs/monitoring-dashboard.md`).

## Icebox

- Non-Pokémon TCG modes (only after Pokémon baseline is solid).
- Full marketplace escrow (out of scope for core product).
