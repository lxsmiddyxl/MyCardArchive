# MCA v1.1.0-beta — economy & ecosystem summary

This marker closes the **v1.1.0-beta** arc: qualitative marketplace negotiation, trust-aware social ranking, creator long-form showcases, and mobile offline polish — **without** real-money checkout or currency pricing.

## Marketplace v3 offers (Phase 81)

- Structured offer DTOs: `offerId`, `fromUserId`, `toUserId`, `itemsOffered`, `itemsRequested`.
- `POST /api/market/v3/offers/create` and `PATCH /api/market/v3/offers/respond` (accept / decline / counter), rate-limited.
- Read-only `GET /api/profile/[id]/offer-history` for negotiation history on profiles.

## Trade rooms v2 (Phase 82)

- Trade room model: `roomId`, `participants`, `messages`, `offers`.
- Server routes: create, join, sanitized message posting (`market_trade_room_messages`).
- `TradeRoomTimeline` UI component for qualitative negotiation timelines.

## Value signals v2 (Phase 83)

- Non-monetary **interest velocity** and **trade desirability** signals.
- `GET /api/market/v3/value-signals` with UI hints: high interest, steady, low activity.

## Collection strength (Phase 84)

- Qualitative categories: **emerging**, **established**, **elite** (no currency).
- `GET /api/profile/[id]/collection-strength` and profile `CollectionStrengthPanel`.

## Creator tools v2 (Phase 85)

- Long-form showcase body (sanitized markdown-style plain text).
- Read-only showcase version history via `showcase_version_snapshots`.
- `ShowcaseLongFormPage` for long-form showcase reading.

## Community v3 (Phase 86)

- `thread_id` on community posts; machine prefix `[mca:thread:…]` for compatibility.
- `POST /api/community/posts/react` (rate-limited).
- `POST /api/community/threads/[threadId]/moderate` for author thread actions.

## Reputation v4 (Phase 87)

- Trust edges `userId → userId` (weighted).
- `trustScoreV4` blends reputation v3, follow graph, and trade reliability.
- `GET /api/profile/[id]/trust-graph`.

## Feed ranking v5 (Phase 88)

- `rankFeedItemsV5` layers trust, interest velocity, and showcase engagement on v4.
- Wired into `GET /api/feed`.

## Mobile v4 (Phase 89)

- Sync Center: offline list LKG panel, gesture affordances (button-driven), clear cache.
- `list-lkg-cache` helpers for feed, binders, decks, and market offers.

## Release

- Package version: **1.1.0-beta**
- Annotated tag: **v1.1.0-beta**
